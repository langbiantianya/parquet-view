# GUI 表格复制功能使用指南

## 功能概述

Parquet View GUI 现在支持强大的表格数据复制功能，可以方便地将数据复制到剪贴板，并粘贴到 Excel、文本编辑器等其他应用程序。

## 支持的复制方式

### 1. 快捷键复制（Ctrl+C）

**使用方法：**
1. 在表格中选择一个或多个单元格
2. 按下 `Ctrl+C`（Windows/Linux）或 `Cmd+C`（Mac）
3. 数据已复制到剪贴板

**特性：**
- 单个单元格：复制单元格的值
- 多个单元格：以 TSV（Tab-Separated Values）格式复制，可直接粘贴到 Excel
- 自动保持行列结构

**示例：**
```
选择的单元格：
  | Name  | Age | City     |
  |-------|-----|----------|
  | John  | 30  | New York |
  | Alice | 25  | London   |

复制后的内容（TSV格式）：
John	30	New York
Alice	25	London
```

### 2. 右键菜单复制

在表格中右键点击，会显示包含以下选项的菜单：

#### 选项 1: **复制** 
复制选中的单元格（与 Ctrl+C 相同）

#### 选项 2: **复制整行**
复制选中行的完整数据，包括表头

**示例：**
```
选择第1行和第2行，右键 → 复制整行

复制内容：
Name	Age	City	Country
John	30	New York	USA
Alice	25	London	UK
```

#### 选项 3: **复制列名**
只复制选中单元格所在列的列名（去重并排序）

**使用场景：**
- 快速获取字段名列表
- 编写 SQL 查询时需要列名

**示例：**
```
选择多个单元格（跨越 Name、Age、City 列）
右键 → 复制列名

复制内容：
Age	City	Name
```

#### 选项 4: **复制所有数据**
复制当前页面的所有数据，包括表头

**注意：**
- 只复制当前页显示的数据，不包括其他分页
- 包含完整的表头信息
- 适合导出当前筛选结果

### 3. 单元格选择模式

GUI 已启用**单元格选择模式**，支持：
- 单选：点击单个单元格
- 多选：按住 Ctrl 键点击多个单元格
- 范围选择：点击起始单元格，按住 Shift 键点击结束单元格
- 行选择：点击行号（最左侧）选择整行
- 列选择：Ctrl + 点击列标题选择整列

## 使用场景

### 场景 1: 复制单个字段值

**操作：**
1. 点击要复制的单元格
2. 按 Ctrl+C
3. 粘贴到目标位置

**适用于：**
- 复制 ID、文件路径、URL 等单个值
- 快速提取特定数据

### 场景 2: 导出数据到 Excel

**操作：**
1. 选择要导出的数据区域（或右键 → 复制所有数据）
2. 按 Ctrl+C
3. 打开 Excel
4. 粘贴（Ctrl+V）

**结果：**
- 数据保持原有的行列结构
- 可以直接在 Excel 中进一步分析

### 场景 3: 获取 SQL 字段列表

**操作：**
1. 选择包含所需字段的任意单元格
2. 右键 → 复制列名
3. 粘贴到 SQL 编辑器

**示例：**
```sql
-- 从剪贴板粘贴列名
SELECT id, name, age, city, email
FROM users
WHERE ...
```

### 场景 4: 批量复制过滤结果

**操作：**
1. 输入 WHERE 条件并应用过滤
2. 等待过滤完成
3. 右键表格 → 复制所有数据
4. 将结果粘贴到 Excel 或文本文件

**适用于：**
- 导出符合条件的数据子集
- 与他人分享筛选结果

### 场景 5: 比较不同行的数据

**操作：**
1. 选择多行的相同列
2. 按 Ctrl+C
3. 粘贴到文本编辑器
4. 比较差异

## 数据格式

### TSV 格式（Tab-Separated Values）

复制的数据使用 TSV 格式，每个字段之间用制表符（Tab）分隔，每行之间用换行符分隔。

**优点：**
- Excel 和 Google Sheets 原生支持
- 纯文本格式，兼容性好
- 保留字段中的空格和特殊字符

**示例：**
```
Name	Age	City	Score
John	30	New York	95.5
Alice	25	London	87.3
Bob	35	Tokyo	92.1
```

### 处理特殊字符

- **空值（NULL）**：显示为空字符串
- **换行符**：保留在字段值中（Excel 会正确处理）
- **制表符**：字段值中的制表符会被保留

## 实用技巧

### 技巧 1: 快速复制大量数据

如果需要复制大量数据（超过当前页），可以：
1. 调整每页行数（右上角的下拉框）为更大的值（如 10000）
2. 加载数据
3. 复制所有数据

### 技巧 2: 配合过滤使用

```
步骤：
1. 输入 WHERE 条件: status = 'active' AND age > 25
2. 点击"应用过滤"
3. 等待过滤完成
4. 右键 → 复制所有数据
5. 得到所有符合条件的数据
```

### 技巧 3: 选择性复制列

只复制需要的列：
1. 按住 Ctrl 键
2. 点击第一列的单元格
3. 向下拖动选择该列的所有可见行
4. 继续按住 Ctrl 点击其他列
5. 按 Ctrl+C 复制

### 技巧 4: 复制到 Markdown 表格

虽然直接复制是 TSV 格式，但可以：
1. 复制数据到剪贴板
2. 粘贴到支持 TSV 转 Markdown 的在线工具
3. 或使用文本编辑器的查找替换功能：
   - 查找: `\t` → 替换: ` | `
   - 在每行前后添加 `|`

## 状态反馈

复制操作完成后，界面底部的状态栏会显示反馈信息：

- `已复制 5 个单元格到剪贴板` - 单元格复制成功
- `已复制 10 行到剪贴板` - 整行复制成功
- `已复制 3 个列名到剪贴板` - 列名复制成功
- `已复制所有 1000 行数据到剪贴板` - 所有数据复制成功
- `复制失败: xxx` - 复制出错

## 常见问题

### Q: 为什么我无法选择单元格？

**A:** 确保：
1. 表格已加载数据
2. 点击的是表格内容区域（不是表头）
3. 如果问题持续，尝试重新加载文件

### Q: 复制后粘贴到 Excel 格式错乱？

**A:** 
1. 确保使用的是 Ctrl+V 粘贴（不要使用"选择性粘贴"）
2. 确保 Excel 识别了 TSV 格式
3. 如果还是有问题，可以先粘贴到文本编辑器查看原始内容

### Q: 能否复制所有分页的数据？

**A:** 
目前"复制所有数据"只复制当前页。如果需要复制多页：
1. 调大每页行数（如设置为 10000）
2. 或者使用过滤功能配合索引读取
3. 或者使用 CLI 版本导出完整数据

### Q: 复制的数据量有限制吗？

**A:** 
理论上没有硬性限制，但受系统剪贴板大小限制：
- Windows: 通常支持几MB
- Mac/Linux: 通常支持更大的数据
- 建议单次复制不超过 100,000 个单元格

### Q: 如何复制包含 NULL 值的数据？

**A:** 
NULL 值会被显示为空字符串。粘贴到 Excel 后会显示为空单元格。
